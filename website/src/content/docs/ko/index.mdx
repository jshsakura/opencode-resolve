`opencode-resolve`는 OpenCode에 집중된 `resolver -> coder` 루프를 추가합니다. 기본 경로는 단순하게 유지하고, 탐색/리뷰는 읽기 전용 서브에이전트가 필요할 때만 맡습니다.

## 빠른 시작

```sh
npm install -g opencode-resolve
opencode plugin opencode-resolve --global --force
opencode
```

그다음 OpenCode에서 `resolver`를 선택합니다.

## 추천 스킬

도메인별 작업에는 [awesome-opencode-skills](https://github.com/jshsakura/awesome-opencode-skills)를 같이 사용해 보세요.

```sh
curl -sL https://raw.githubusercontent.com/jshsakura/awesome-opencode-skills/main/install.sh | bash
```
